Detailed Description

parse-ms

Parse milliseconds into an object

Install

npm install parse-ms

Usage

import parseMilliseconds from 'parse-ms';
parseMilliseconds(1337000001);
/*
{
days: 15,
hours: 11,
minutes: 23,
seconds: 20,
milliseconds: 1,
microseconds: 0,
nanoseconds: 0
}
*/
parseMilliseconds(1337000001n);
/*
{
days: 15n,
hours: 11n,
minutes: 23n,
seconds: 20n,
milliseconds: 1n,
microseconds: 0n,
nanoseconds: 0n
}
*/
